# Yasaflow Product Vision

## Vision

Yasaflow should become a leading digital platform for organizations that need one clear place for members, activities, communication, administration and future payments.

## Core value

Organizations should not need many disconnected tools to run their daily work.

Yasaflow brings together:

- News
- Activities
- Members
- Push notifications
- Donations
- Documents
- Calendar
- Chat
- Administration
- Settings

## User groups

- Yasaflow Owner: manages the platform.
- Organization Administrator: manages one organization.
- Member: belongs to an organization and uses its services.

These roles must not be mixed.

## Website purpose

The website should help visitors understand Yasaflow in less than 60 seconds, see whether it fits their organization and choose a clear next step.

## Phases

1. Public marketing website.
2. Owner-approved public onboarding request.
3. Public self-service onboarding.
4. Packages, payments and automated provisioning.
