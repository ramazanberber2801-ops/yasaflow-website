# Changelog

- Header, hero, organization types, features, setup timeline, product previews, CTA and footer.

- Refined the hero copy, visual hierarchy and Yasaflow blue/teal brand accents.
- Increased organization-card icon presence.
- Localized navigation, core sections, calls to action and footer content.
