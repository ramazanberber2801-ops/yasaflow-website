# Changelog

All meaningful changes to the Yasaflow public website are recorded here.

## 2026-07-11

### Added
- Typed central module catalog with permanent IDs and visibility rules.
- Localized module metadata and shared icon resolver.
- Multilingual About, Contact, Privacy and Terms pages.
- Lightweight public routing, Vercel rewrites and a 404 fallback.
- Route- and language-specific SEO metadata.
- Canonical URLs, Open Graph and Twitter Card metadata.
- Organization structured data.
- `robots.txt`, `sitemap.xml`, web app manifest and favicon.

### Changed
- Replaced homepage hardcoded module lists with `getPublicModules()`.
- Distinguished included modules from optional extensions without exposing prices.
- Connected homepage navigation to dedicated public pages.
- Marked unknown routes as `noindex, nofollow`.

## 2026-07-10

### Added
- First production-quality public website UI.
- Project playbook and website documentation standards.
- Dedicated “Why Yasaflow?” section.
- Multilingual website support for English, Norwegian and Turkish.
- Automatic browser-language detection and persistent manual language selection.
- Official Yasaflow v1 logo icon asset.

### Changed
- Refined the hero copy, visual hierarchy and Yasaflow brand accents.
- Localized navigation, core sections, calls to action and footer content.
- Replaced the temporary letter placeholder with the official Yasaflow logo design in the reusable Logo component.
