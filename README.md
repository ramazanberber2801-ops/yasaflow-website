# Yasaflow Website

Public website and future onboarding portal for Yasaflow.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Lucide React

The project is prepared for Framer Motion, but animation dependencies are intentionally postponed until the design phase.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
